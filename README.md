**Basic Calculator**
A clean and simple calculator built with HTML, CSS, and JavaScript that performs basic arithmetic operations right in your browser.
**Live Demo**
https://shreyanshshekhar17.github.io/basic-calculator-in-js/calculator/
**Features**
Addition, Subtraction, Multiplication, Division
Clean and minimal UI
Instant results — no page reload
Works on all modern browsers
**Tech Stack**
HTML        Structure/Layout
CSS          Styling
JavaScript   Logic&Calculations
